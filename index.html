Основы HTML таблиц

HTML таблицы создаются с помощью тега <table>.Основные элементы таблицы:

  • <tr> - строка таблицы
  • <th> - заголовок таблицы
  • <td> - ячейка таблицы
<table>
  <tr>
    <th>Заголовок 1</th>
    <th>Заголовок 2</th>
  </tr>
  <tr>
    <td>Данные 1</td>
    <td>Данные 2</td>
  </tr>
</table>

Расширенные возможности таблиц

Объединение ячеек

Для объединения ячеек используются атрибуты colspan и rowspan.

<table>
	  <tr>
	    <th colspan="2">Объединенный заголовок</th>
	  </tr>
	  <tr>
	    <td rowspan="2">Объединенная ячейка</td>
	    <td>Ячейка 1</td>
	  </tr>
	  <tr>
	    <td>Ячейка 2</td>
	  </tr>
	</table>

Семантические элементы таблиц

Для лучшей структуризации таблицы можно использовать дополнительные теги:

  • <thead> - заголовок таблицы
  • <body> - подвал таблицы
  • <caption> - заголовок таблицы
<table>
	  <caption>Ежемесячные продажи</caption>
	  <thead>
	    <tr><th>Месяц</th><th>Продажи</th></tr>
	  </thead>
	  <tbody>
	    <tr><td>Январь</td><td>150 000 ₽</td></tr>
	    <tr><td>Февраль</td><td>180 000 ₽</td></tr>
	  </tbody>
	  <tfoot>
	    <tr><td>Итого</td><td>330 000 ₽</td></tr>
	  </tfoot>
	</table>
